Hive MQ

HiveMQ is a MQTT broker, specifically for enterprises which find themselves in the emerging age of Machine-to-Machine communications (M2M) and the Internet of Things. It was built from the ground up with maximum scalability, easy management and security in mind. We provide our customers the ability to connect to HiveMQ with a completely secure environment.

Cloud Foundry

The M2M Dashboard is built on top of Cloud Foundry which is an open source cloud computing platform as a service (PaaS). It supports applications in Java, Ruby, Javascript, Spring and many more. Cloud Foundry provides built-in services which can be bound to the application such as Redis, RabbitMQ, MySQL, PostgreSQL, MongoDB and many more.

ClearDB MySQL

We keep you credentials safe. ClearDB is a higly available MySQL service for all applications. ClearDB provides a secured database because all the data can be encrypted. It allows the database to scale indefinitely with any number of replicas that are needed.
